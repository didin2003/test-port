Project information

  • Name: Job Portal
  • Category: Web application
  • Tools: Github, Jenkins, SonarQube, Trivy, Docker, Prometheus, Grafana, Kubernetes (ArgoCD)
  • CloudPlatform: AWS

Description

I successfully deployed a comprehensive MERN stack job portal application utilizing a modern CI/CD workflow that integrates GitHub, Jenkins, Docker, SonarQube, and ArgoCD. The source code is managed in GitHub, while Jenkins automates the build, test, and containerization processes, producing Docker images for consistent deployment. Code quality and security are continuously analyzed using SonarQube, ensuring robust and maintainable code. The application is deployed across multiple regions to ensure high availability and low latency, with ArgoCD managing the GitOps-based deployment pipelines to synchronize the infrastructure and application state automatically. This multi-region, containerized setup allows for seamless updates, scalability, and resilient delivery of the job portal services.
