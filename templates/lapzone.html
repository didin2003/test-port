Project information

  • Name: Lapzone
  • Category: Web application
  • Tools: Github, Jenkins, SonarQube, Trivy, Docker, Prometheus, Grafana, Kubernetes (ArgoCD)
  • CloudPlatform: AWS

Description

I successfully launched LapZone – a Laptop Sales, Service & Rentals platform on AWS using a multi-tier architecture with EC2 instances (t3.medium for frontend, t3.large for backend + database) across eu-north-1 and us-east-1, fully integrated with Dockerized services for consistent deployments, an automated Jenkins CI/CD pipeline for seamless builds and releases, and Kubernetes NodePort service exposure for external access, where the platform delivers a complete e-commerce-style UI enabling users to browse, rent, or purchase laptops and request services, while the deployment strengthened my expertise in Kubernetes networking, cloud-native infrastructure, DevOps practices, and scalable application hosting.
